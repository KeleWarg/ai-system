# AI Design System Components

Installable package of AI-generated React components.

## Installation

```bash
npm install ./dist-package
```

Or copy the `components` folder to your project.

## Usage

```tsx
import { Button } from '@ai-design-system/components'

<Button variant="primary">Click me</Button>
```

## Components

- **Button** (`button`) - Interactive button component with multiple variants, states, and sizes for user actions

## Required Dependencies

- @radix-ui/react-slot
- class-variance-authority
- clsx
- tailwind-merge
- react@^18.0.0
- @radix-ui/react-slot@^1.0.0
- class-variance-authority@^0.7.0
